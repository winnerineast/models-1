## Description
ResNet-50 is a deep convolutional networks for classification.

### Paper
[Deep Residual Learning for Image Recognition](https://arxiv.org/abs/1512.03385)

### Dataset
[ILSVRC2015](http://www.image-net.org/challenges/LSVRC/2015/)

### Demo
